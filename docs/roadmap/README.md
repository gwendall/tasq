# Roadmap

[`BACKLOG.json`](BACKLOG.json) is the machine-readable authority for order,
dependencies, status, and external gates. [`BACKLOG.md`](BACKLOG.md) explains
the same sequence for humans.

A detailed implementation and acceptance specification for the current public
adoption, SDK, Server, Cloud, offline-replication and cross-language work is in
[`PUBLIC_ADOPTION_TO_CLOUD_EXECUTION_PLAN.md`](PUBLIC_ADOPTION_TO_CLOUD_EXECUTION_PLAN.md).

A planned or candidate item does not change current support claims. Those are
owned by [`../concepts/PRODUCT_SURFACE_MATRIX.json`](../concepts/PRODUCT_SURFACE_MATRIX.json).
